Amoeba Talk create order as a Customer for WooCommerce as a Customer allows store administrators to login as a customer on the frontend.

== Description ==

Here comes a simple plugin that brings about the greatest time-saving benefits for shop owners and store managers. This plugin offers an easy and quick switch from the administrator role to any other role without the need to request a password from users/customers. It's a life-saving tool for shop owners to view their shop from a different role or switch between different accounts easily with just 1 click.

Logged in as a customer allows store owners to check the ordering system and optimizing the shopping experience for better performance. Stayed logged in as long as you want to double-check orders, changing as many roles as you want, and switch back in can be done within a few clicks right from the front end.

= Key Features =

- Link to talk amoeba
- Choose user roles can use this feature
- Choose user roles that they can switch to
- AJAX search users and login selected user
